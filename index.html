Self Hosted App Starter

A starter for the self-hosted app, help you to build your next full-stack project.

Getting Started →

🪄 Full-stack

Full-stack starter with GSVT Stack, which stands for Go、SQLite、Vue 3、TailwindCSS

📦 Back-end

Using Go & SQLite as the Back-end Tech Stack, Using Air for Live-reloading Go apps

🖖 Front-end

Choose your familiar Frontend Tech Stack, default is Vue 3, and it served by Vite

🐙 Open Source

Fully open-source on the GitHub, Fork it, Ship it, Self-Hosted it.

⚓️ Self-hosted

Self-hosted with `Dockerfile` on the fly, docker image ready to use.

🚄 Deploy Automatically

Deploy you App on the Railway as easy as 1, 2, 3
